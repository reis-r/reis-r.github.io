New computer, strange freezes and silly solution

29/07/2020
By Rony Reis
Category: misc
Tags: personal computers

After enduring for 7 years, my Lenovo Ideapad S400 started to give up on me. It was my main and mostly only machine. I wished to buy a Thinkpad as an upgrade since I liked it so much, but with the whole COVID-19 crisis, the prices skyrocketed. I ended up buying an Acer Aspire 3 with AMD processor/video card, it was a more cost-effective choice than another Ideapad. I'm quite happy with it so far, and I'm enjoying being able to play some modern games.

But as soon as I got everything setup on the new machine, it started to freeze randomly. It would happen on random times, usually when I used a lot of RAM. It happened a lot of times when I was using Jupyter Kernels, and mostly when changing context from one application to another. The mouse wouldn't move, sound would glitch, all the system froze. I came to the point of reinstalling Windows 10 altogether two times, trying different driver versions and software to troubleshot the problem. I was praying for it not to be a factory defect since I could not afford not having a computer for work and study. The solution was quite surprising for me...

Two months of intensive testing and freezing and rebooting passed. I tried changing habits and not opening lots of programs and keeping them opened as I did previously. But what would be the point of having a newer, faster machine, then?

And then I searched for something I was sure it was NOT the problem, since I had used for my entire life and it has never failed me: Veracrypt. It turns out Veracrypt was just the cause of the problem. I had always used it's full disk encryption on Windows, but I used Windows 8 previously, and looks like with newer versions of Windows 10 it has a bug where the encryption keys end up in the paging file and deadlocking the system. You can find about it clicking on this thread.

So the solution was disabling full system encryption, encrypting only sensitive files (and using Window's native system encryption). Of course, I could always go back to Linux, or dual booting, but I'm enjoying what became of Windows 10. I'll sure write about my experience on another posts.
